# CRANKD
**The Digital Soul of Your Machine.**

CRANKD is a high-fidelity system of record for vehicle ownership, maintenance, and heritage. It decouples the vehicle's digital identity from the temporary owner, creating a permanent, value-accreting asset.

## Project Structure

-   `mobile/`: React Native (Expo) application. The primary user interface.
-   `backend/`: Supabase configuration, SQL schemas, and Edge Functions.
-   `web/`: Next.js marketing site and Admin Dashboard.

## The Stack

-   **Client:** React Native (Expo), Skia (Graphics), Reanimated anywhere.
-   **Backend:** Supabase (PostgreSQL, Auth, Realtime).
-   **Media:** Cloudflare R2 + Imazu.
-   **AI:** Google Document AI (OCR).
